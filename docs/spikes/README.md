This directory holds engineering spikes for scoped Inkwell design investigations.

- **001** — Spike 001: Browser Login UI — Designs a flag-gated author token login page that exchanges scoped tokens for browser sessions without adding admin login or production code.
- **002** — Spike 002: Write-Audit History API — Defines a read-only document history endpoint that exposes append-only write-audit events to admins and document owners.
