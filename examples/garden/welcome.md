# Welcome to Your Garden

This is a **digital garden** seeded by `docker compose up`. Unlike a blog, a
garden is a web of interlinked notes that grow and connect over time.

Start exploring:

- [[what-is-a-digital-garden]] — the idea behind this whole thing.
- [[wikilinks-and-backlinks]] — how notes connect to one another.
- [[reading-with-ai]] — let an AI agent read, search, and edit these notes.
- [[bring-your-own-vault]] — import your existing Obsidian notes.

Every link above is a `[[wikilink]]`. Open any note and scroll to the
**Linked from** panel at the bottom to see which other notes point back to it.
